Emotion-Detection

Emotion-Detection Description This project detects human emotions from text and speech data using advanced Natural Language Processing and deep learning techniques. The system is trained to understand emotional context in written text as well as vocal expressions.

Key features include:

A text dataset of emotional samples cleaned and prepared from Reddit (used for training the text emotion model).

A model built with BERT (a transformer-based NLP model) to classify emotions accurately from text input.

An interactive Python application (app.py) that can take user input and display predicted emotions.

Use of common Python libraries such as Pandas for data handling and NLP tools to preprocess text.

The project includes both the ML model training notebook (model.ipynb) and application code.
